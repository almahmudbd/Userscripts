# Universal Dark Mode 3 (Gray-Blue)

## Description

Universal Dark Mode 3 is a userscript that applies a gray-blue dark mode across all websites, enhancing readability during night-time browsing by dynamically adjusting background, text, and border colors.

## Installation

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://www.greasespot.net/).

2. Create a new script and replace the default code with the code from this repository.

3. Save the script to activate dark mode.
